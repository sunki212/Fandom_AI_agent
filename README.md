# Fandom_AI_agent
Учебный проект — агент на базе LLM для генерации идей и ответов по вселенным Fandom Wiki

# Что это?  
Fandom AI Agent — учебная демонстрация агентной системы, которая:  
🎲 Генерирует идеи для квизов по Гарри Поттеру, Марвел, Властелину колец  
💡 Подкидывает фанатские теории и пересечения мультивселенных  
🔍 Ищет информацию в Fandom Wiki (иногда выдумывает креативные ответы)  
🧠 Показывает, как работают LLM-агенты с планированием и инструментами  
Важно: Проект создан в образовательных целях. Ответы могут быть вымышленными или неточными — используйте для вдохновения и развлечения! 

# ✨ Возможности  
* 9 вселенных: Harry Potter, Marvel, LOTR, Star Wars, Game of Thrones, Witcher, Elder Scrolls, Warcraft, DC  
* Интерактивный UI с историей запросов  
* Кэширование ответов (повторные вопросы — мгновенно)  
* Метрики и логирование  

# 🚀 Быстрый старт в Google Colab  
1. Откройте Colab → Среда выполнения → Сменить среду выполнения → T4 GPU  
2. Выполните ячейку 1 (установка библиотек)  
3. Перезапустите сеанс (Среда выполнения → Перезапустить сеанс)  
4. Выполните ячейки 2 и 3 (загрузка модели и запуск UI)

# 📊 Выводы
**Что получилось**  
* Агент успешно планирует поисковые запросы через LLM  
* Кэширование ускоряет повторные ответы в 50+ раз  
* Определение вселенной по ключевым словам работает с точностью ~95%  
* UI удобен для экспериментов и демонстрации

**Что не идеально**
* Qwen2.5-1.5B иногда галлюцинирует (выдумывает факты)  
* При одном источнике ответы слишком краткие  
* Нет поддержки эмбеддингов для семантического поиска  
* Требует GPU (T4)

# 🛠 Технологии
* LLM: Qwen2.5-1.5B-Instruct (HuggingFace Transformers)  
* Поиск: fandom-py + custom детекция вселенных  
* Метрики: Prometheus client  
* UI: ipywidgets    
* Оптимизация: bitsandbytes (float16)

# Структура проекта
